# Gowda Connect

A Django-based web application for managing directories and user services.

## Features
- User Registration & Login
- Directory Listing
- Contact & Services Pages
- Admin Panel

## Tech Stack
- Python
- Django
- HTML, CSS

## How to Run
1. Clone the repo
2. Install requirements
3. Run: python manage.py runserver
4. Open http://127.0.0.1:8000/
